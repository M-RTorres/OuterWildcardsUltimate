	- ABOUT -

Outer Wildcards is a Texture Pack for Balatro, themed around Outer Wilds

This pack comes in 2 versions:
- The Full version, which contains all the available changes
- A Reduced version, which aims to be less intrusive and focus more centrally on planet cards

This is the Full version of the pack

There will also be a third release for Malverk compatability, but that is still a work in progress (sorry)
Be sure to follow the itch.io page to be notified when that releases


	- CHANGES -

The following are only the changes included in both pack versions:

- Each planet card (including black hole) has been changed to an astral body from Outer Wilds
- The Celestial Packs have been changed to feature art of the Travellers
- Custom Outer Wilds club face cards and ace have been added

The following are only the changes included in only the Full version:

- Celestial Packs have been renamed to Wilderness Packs
- Meteor Tags have been changed to Campfire Tags
- Subtle tweaks have been made to High Priestess, Trance, Blue Seal, and Orbital Tag
- The Vouchers Telescope and Observatory have been resprited
- The following jokers have been resprited:
	Supernova, Space Joker, Constellation, Rocket, Obelisk, Campfire, Satellite, Astronomer, Burnt Joker


	- INSTALLATION -

This pack requires Steamodded

- If you have not already, install Steamodded [https://github.com/Steamopollys/Steamodded]
- Extract your OuterWildcards.zip
- Move this directory (Outer Wildcards (full)) into your Mods folder [%appdata%/Balatro/Mods]
